#### Development

``` bash
# install dependencies
npm install
-or-
yarn

# serve with hot reload at localhost:9080
npm run dev
-or-
yarn dev

# build electron application for production
npm run build
-or-
yarn build

# lint all JS/Vue component files in `src/`
npm run lint
-or-
yarn lint

```
